 #Fake News Detection AI

AI-powered fake news detection system built using Python, Machine Learning, and Streamlit.

## Features
- Fake/Real news prediction
- Streamlit UI
- Model accuracy display
- Interactive text input

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- TF-IDF Vectorizer
- Logistic Regression / Naive Bayes

## Run Project
pip install -r requirements.txt
streamlit run app.py
